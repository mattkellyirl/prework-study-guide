Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for cascading style sheet
  • Margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside of an element.

Git

  • Git is version control
  • git status: This git command checks what branch we are currently on.
  • git checkout -b branch-name: This git command creates a new branch and switches to it.

Javascript

  • Javascript enables functionality and interation with web applications.
